# Xplorer — macOS向けWindows Explorer風ファイラー 設計書

## 概要

macOS上でWindowsエクスプローラーと同等のUI/UXを再現するファイルマネージャ。Tauri v2 + React + TypeScript + Rustで構築する。

---

## 技術スタック

| レイヤー | 技術 | 理由 |
|---------|------|------|
| フレームワーク | Tauri v2 | 軽量、ネイティブWebView、Rust backend |
| フロントエンド | React 18 + TypeScript | AI生成精度が高い、エコシステム充実 |
| スタイリング | CSS Modules or Tailwind | コンポーネント単位のスタイル管理 |
| バックエンド | Rust (Tauri Commands) | ファイルI/O高速処理、権限管理 |
| 状態管理 | Zustand | 軽量、ボイラープレート少 |
| アイコン | lucide-react + カスタムSVG | 拡張子別アイコンマッピング |

## コンポーネント設計

### 1. TitleBar（タイトルバー + タブ）

```
コンポーネント: <TitleBar />
子コンポーネント: <TabBar />, <TabItem />
```

- Tauriのデコレーション無効化 → カスタムタイトルバー
- タブ: Chrome風ドラッグ可能タブ
- 各タブが独立したパス状態を持つ
- タブの追加/閉じる/ドラッグ並替/中クリックで閉じる
- data-tauri-drag-region でウィンドウドラッグ対応

### 2. NavigationBar（ナビゲーションバー）

```
コンポーネント: <NavigationBar />
子コンポーネント: <NavButtons />, <AddressBar />, <SearchBox />
```

#### NavButtons
- 戻る（←）: 履歴スタックのpop
- 進む（→）: forward履歴スタック
- 上へ（↑）: parent directory
- 履歴ドロップダウン（戻る/進む長押し）

#### AddressBar（最重要コンポーネント）
- **通常モード**: パンくずリスト表示（クリックで各階層に移動）
- **編集モード**: クリックでテキスト入力に切替、パス直打ち可能
- パンくずの各セグメント右に「＞」ボタン → 兄弟ディレクトリのドロップダウン
- Enter で移動、Escape でキャンセル
- パス補完（Tab キー）

#### SearchBox
- 現在ディレクトリ内のインクリメンタルサーチ
- Rust側でglob/regex検索実行
- 検索結果をメインペインに表示

### 3. Toolbar（ツールバー）

```
コンポーネント: <Toolbar />
```

- Windowsのリボンは不要。シンプルなアイコンツールバー
- アクション: 新規フォルダ, 切取, コピー, 貼付, 名前変更, 削除, ゴミ箱
- 表示切替: 詳細 / リスト / アイコン（小/中/大）
- ソート: 名前, 更新日, サイズ, 種類
- 選択状態に応じてボタンのenable/disable

### 4. SidePanel（ナビゲーションペイン）

```
コンポーネント: <SidePanel />
子コンポーネント: <SideSection />, <TreeNode />
```

- **クイックアクセス**: ピン留めフォルダ（Desktop, Downloads, Documents, Pictures等）
- **PC**: マウントされたボリューム一覧
- **ツリービュー**: ディレクトリの展開/折畳（遅延読込）
- ドラッグ&ドロップでクイックアクセスにピン追加
- リサイズ可能（ドラッグで幅調整）
- 右クリックでコンテキストメニュー

### 5. MainPane（メインペイン）

```
コンポーネント: <MainPane />
子コンポーネント: <DetailView />, <ListView />, <IconView />, <FileItem />
```

#### DetailView（詳細表示 — デフォルト）
- カラム: 名前, 更新日時, 種類, サイズ
- カラムヘッダークリックでソート（昇順/降順トグル）
- カラム幅ドラッグリサイズ
- カラム右クリックで表示/非表示切替

#### ListView（リスト表示）
- ファイル名のみ、複数カラムレイアウト

#### IconView（アイコン表示）
- 小/中/大/特大アイコン
- サムネイルプレビュー（画像ファイル）

#### 共通動作
- クリック: 選択
- Ctrl+クリック: 複数選択
- Shift+クリック: 範囲選択
- ダブルクリック: 開く（フォルダ→移動, ファイル→デフォルトアプリ）
- ドラッグ: 移動/コピー（Optionキーでコピー）
- 矩形選択（ラバーバンド）
- F2: 名前変更（インライン編集）

### 6. ContextMenu（コンテキストメニュー）

```
コンポーネント: <ContextMenu />
```

- 右クリックで表示
- 選択対象に応じてメニュー項目を動的生成
- **ファイル選択時**: 開く, 新しいタブで開く, 切取, コピー, パスをコピー, 削除, 名前変更, プロパティ
- **フォルダ選択時**: 上記 + ターミナルで開く
- **空白右クリック**: 新規フォルダ, 新規ファイル, 貼付, 表示切替, ソート, プロパティ
- キーボードショートカット表示
- セパレータ対応

### 7. StatusBar（ステータスバー）

```
コンポーネント: <StatusBar />
```

- 項目数表示
- 選択中の項目数とサイズ合計
- 表示切替ボタン（アイコン）

### データ構造

```rust
#[derive(Serialize)]
struct FileEntry {
    name: String,
    path: String,
    is_dir: bool,
    size: u64,
    modified: i64,          // Unix timestamp
    created: i64,
    file_type: String,      // 拡張子 or "folder"
    is_hidden: bool,
    is_symlink: bool,
    permissions: String,
}

#[derive(Serialize)]
struct FileMetadata {
    #[serde(flatten)]
    entry: FileEntry,
    mime_type: Option<String>,
    owner: String,
    group: String,
}

#[derive(Serialize)]
struct VolumeInfo {
    name: String,
    mount_point: String,
    total_space: u64,
    available_space: u64,
    fs_type: String,
}
```

## スタイリング方針

### Windowsエクスプローラー再現のポイント

1. **色味**: Windows 11準拠
   - 背景: `#FFFFFF`（メインペイン）, `#F3F3F3`（サイドパネル）
   - アクセント: `#005FB8`（選択）
   - テキスト: `#1B1B1B`
   - ボーダー: `#E5E5E5`
   - ホバー: `#F5F5F5`
   - 選択行: `#CCE4F7`

2. **フォント**: Segoe UI → macにないので `"Segoe UI", -apple-system, sans-serif`

3. **角丸**: Windows 11スタイルの `4px` 角丸

4. **アイコン**: Fluent UI System Icons（Microsoft公式、MITライセンス）
   - `@fluentui/svg-icons` から使用

5. **アニメーション**: 最小限。ホバーの opacity transition 程度

6. **ダークモード**: 対応する（Windowsエクスプローラー準拠のダークテーマ）
   - 背景: `#202020`（メイン）, `#2D2D2D`（サイド）
   - テキスト: `#FFFFFF`
   - ボーダー: `#3D3D3D`
